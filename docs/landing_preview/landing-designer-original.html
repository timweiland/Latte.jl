From model to fit,
in one pour.

Specify a spatial Poisson GLMM, fit it with a Laplace approximation, then flip to full MCMC when the reviewer asks. Same model object — no re-writing, no re-parametrizing.

spatial_glmm.jl
model
fit
summary
# Disease mapping — counts on a Besag graph
model = @latte begin
  β  ~ Normal(0, 10)                 # fixed effects
  κ  ~ Gamma(1, 5e-5)             # precision
  u  ~ GMRF(κ; graph = mesh)     # spatial prior
  y  ~ Poisson(exp(X*β + u + log(E)))
end

# Laplace first — for the dashboard
fit  = brew(model, data; method = :laplace)

# MCMC later — for the paper
post = brew(model, data; method = :mcmc, chains = 4)
